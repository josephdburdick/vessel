# Vessel CMS
##Content Management System Boilerplate for Meteor.

### Installation

- Install Meteor:

```sh
$ curl https://install.meteor.com | /bin/sh
```

- Git clone this repo and cd into it.

```sh
$ git clone https://github.com/josephdburdick/vessel.git
$ cd vessel
```

- Run Vessel:

```sh
$ npm start
```

- Debug Vessel:

```sh
$ npm run-script debug
```
